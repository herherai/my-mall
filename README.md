### i believe i can fly

### 初始化第一步：
在assets文件夹下创建css，依次引入normalizec.css和base.css

### 初始化第二步：.editorconfig
项目由cli4搭建而成，未能自动生成该文件。用于规范文件代码格式，避免因多人开发格式不统一而造成的混乱。

### 初始化第三步：vue.config.js
添加路径别名，避免文件路径层次过多造成引入错误。
